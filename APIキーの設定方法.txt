============================================
  APIキーの設定方法（かんたん版）
============================================

【重要】
.env.local というファイルは、すでに作ってあります！
左の一覧に見えなくても、大丈夫です。

場所：
C:\Users\VAIO\新しいフォルダー\voice-ai-image-studio\.env.local


============================================
  やることは2つだけ
============================================

■ ステップ1：APIキーを取得する

このURLをブラウザで開いてください：
https://platform.openai.com/api-keys

→ ログイン
→ 「Create new secret key」をクリック
→ 表示された sk- で始まる文字列をコピー


■ ステップ2：チャットに貼り付ける

コピーしたAPIキーを、このチャットに貼り付けて送ってください。
こちらで .env.local に入れます。

※ APIキーはあなた専用のパスワードのようなものです。
  他人には見せないでください。


============================================
  設定が終わったら
============================================

ターミナルで：
  npm run dev

ブラウザで開く：
  http://localhost:3000
